## **💡 About Me**

I build developer tools and write the technical content that gets them adopted. Right now I'm **Technical Content Creator at Altimate AI**, writing for data engineers, and I run **Developer Relations and security content at AccuKnox** (the team behind **KubeArmor**). Before that I freelanced as a Python specialist for TutorialsPoint, writing tutorials on Pandas, NumPy, Matplotlib, Flask, and Django.

My work sits where engineering meets communication: CLIs on PyPI, AI automation pipelines, and documentation that ranks and converts.

**🔥 Current Focus:**
- Writing for **data engineers** at Altimate AI (dbt tooling, Power User)
- **AI and cloud security** content and DevRel at AccuKnox
- Shipping **Python CLIs and AI tools** (VocabCLI, musicli, and more)
- Growing **atharvashah.com** into a full content and product brand
